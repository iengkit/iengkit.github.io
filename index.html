A.I. Logics + Stochastic Local Search - SAT Solver (R)

A set of clauses are generated randomly per run for a specified number of runs dependent on N and C. Each numerical digit represents a symbol in clauses are used. Within a specified number of maxflips the program will try to find a model that satisfies the clauses by randomly selecting a clause that is false in the model, randomly chooses a symbol from that false clause then flip the value of that symbol in the model. The program then returns a satisfying model and output the number of flips required to generate the model if there is one within maxflips.
